# Telegram Bot для подтверждения возраста КСО

Бот для подтверждения возраста пользователей на КСО с системой очереди, модерации и балансом.

## Возможности

- Подача заявок на подтверждение возраста (250₽)
- Система очереди с расчетом времени ожидания
- Панель модератора для обработки заявок
- Обмен фотографиями между пользователем и модератором
- Система баланса с пополнением и списанием средств

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в корне проекта:
```
BOT_TOKEN=ваш_токен_бота
```

3. Запустите бота:
```bash
python bot.py
```

## Структура проекта

- `bot.py` - главный файл бота
- `config.py` - конфигурация
- `database/` - модели и работа с БД
- `handlers/` - обработчики сообщений
- `keyboards/` - inline клавиатуры
- `utils/` - утилиты (очередь, баланс)

## База данных

Используется SQLite для хранения:
- Пользователей и их балансов
- Заявок на подтверждение
- Сессий модерации
- Транзакций
- Статистики модераторов
