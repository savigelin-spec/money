## Блок 1. Архитектура и общая картина

### Архитектура системы

Система состоит из следующих компонентов:

- **Telegram бот** (Python + aiogram 3.x)
- **База данных SQLite** для хранения пользователей, заявок, балансов, сессий модерации
- **Система очереди** с расчетом времени ожидания
- **Панель модератора** для управления заявками
- **Система баланса** с пополнением и списанием средств

### Структура файлов проекта

```text
money/
├── bot.py                 # Главный файл бота
├── config.py              # Конфигурация (токен, настройки)
├── database/
│   ├── __init__.py
│   ├── models.py          # SQLAlchemy модели
│   ├── db.py              # Инициализация БД
│   └── queries.py         # Запросы к БД
├── handlers/
│   ├── __init__.py
│   ├── user_handlers.py   # Обработчики пользователей
│   ├── moderator_handlers.py  # Обработчики модераторов
│   └── payment_handlers.py    # Обработчики платежей
├── keyboards/
│   ├── __init__.py
│   ├── user_keyboards.py  # Клавиатуры для пользователей
│   └── moderator_keyboards.py  # Клавиатуры для модераторов
├── utils/
│   ├── __init__.py
│   ├── queue.py           # Логика очереди и расчета времени
│   └── balance.py         # Работа с балансом
├── requirements.txt
└── README.md
```
