## Блок 3. Логика пользователей и модераторов

### Функции для пользователей

1. **Главное меню** (кнопки):  
   - «Подать заявку на подтверждение» (250₽)  
   - «Мой баланс»  
   - «Пополнить баланс»  
   - «Мои заявки»  

2. **Подача заявки**:  
   - Проверка баланса (минимум 250₽)  
   - Списание средств  
   - Создание заявки в очереди  
   - Расчёт позиции в очереди и примерного времени ожидания  
   - Уведомление о создании заявки  

3. **Ожидание модератора**:  
   - Периодические обновления позиции в очереди (по кнопке или по событию)  
   - Уведомление при подключении модератора  

4. **Сессия модерации**:  
   - Пользователь отправляет скриншот в бота  
   - Скриншот пересылается модератору  
   - Модератор присылает фото пользователю  
   - Завершение сессии (статус completed / rejected)  

### Функции для модераторов

1. **Панель модератора** (кнопки):  
   - «Доступные заявки» (список pending)  
   - «Мои активные сессии»  
   - «Статистика»  

2. **Выбор заявки**:  
   - Просмотр списка заявок с позицией в очереди  
   - Ручной выбор заявки для модерации  
   - Уведомление пользователя о подключении модератора  

3. **Процесс модерации**:  
   - Получение скриншота от пользователя  
   - Отправка фотографии пользователю  
   - Кнопки «Подтвердить» / «Отклонить» заявку
