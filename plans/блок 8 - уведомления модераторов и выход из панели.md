# Блок 8: Уведомления модераторов о новых заявках и выход из панели

## Обзор

Добавить систему уведомлений модераторов о новых заявках с автоматическим удалением уведомления после обработки заявки модератором (approve/reject), а также добавить кнопку выхода из панели модератора в главное меню.

## Структура блоков

- **Блок 8.1**: Модель данных и запросы к БД для уведомлений модераторов
- **Блок 8.2**: Утилиты для удаления уведомлений
- **Блок 8.3**: Сохранение уведомлений при отправке
- **Блок 8.4**: Удаление уведомлений при обработке заявки (approve/reject)
- **Блок 8.5**: Кнопка выхода из панели модератора

## Текущее состояние

- Функция `notify_moderators_new_application` отправляет уведомления всем модераторам через `bot.send_message`
- Уведомления не сохраняются в БД и не удаляются после обработки заявки
- В панели модератора нет кнопки для возврата в главное меню
