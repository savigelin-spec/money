# Планы реализации проекта

## Структура планов

### Блоки 1-5: Основной функционал бота
- **блок 1.md** - Описание блока 1
- **блок 2.md** - Структура базы данных
- **блок 3.md** - Описание блока 3
- **блок 4.md** - Описание блока 4
- **блок 5.md** - Описание блока 5

### Блок 6: Система статистики

**Основной файл:**
- **блок 6 - статистика.md** - Обзор системы статистики

**Подблоки:**
- **блок 6.1 - расширение модели данных и UTM.md** - Расширение модели данных и отслеживание трафика (2-3 часа)
- **блок 6.2 - бизнес показатели.md** - Бизнес-показатели (финансы, заявки, пользователи) (4-5 часов)
- **блок 6.3 - маркетинговые показатели.md** - Маркетинговые показатели (конверсия, LTV, CAC, retention) (5-6 часов)
- **блок 6.4 - статистика по источникам трафика.md** - Статистика по источникам трафика (3-4 часа)
- **блок 6.5 - UI для админов и форматирование.md** - UI для админов и форматирование (4-5 часов)
- **блок 6.6 - оптимизация и тестирование.md** - Оптимизация и тестирование (2-3 часа)

## Порядок реализации

Блоки 6.1-6.6 должны реализовываться последовательно, так как каждый блок зависит от предыдущего.
